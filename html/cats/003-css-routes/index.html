2. Примеры селекторов

СЕЛЕКТОР
{

2.1 Простые примеры

.Cup
Точка — специальный символ для класса (class)
Без точки CSS будет искать HTML тег с таким названием
комната №1
.WashingMachine   .Cup
CSS ищет чашки внутри посудомойки.
Пробел означает «искать внутри»
комната №2
.Cup.RedColor
Искать чашки с красным цветом
Составной селектор. Без пробела CSS ищет элемент у которого есть все перечисленные классы
комната №6
.WashingMachine   .Cup.RedColor   .Liquid
Жидкость внутри красной чашки, внутри посудомойки.
Можно использовать несколько пробелов для создания цепочки. Каждое звено цепочки может быть составным селектором.
комната №3
.WashingMachine > .Liquid
Выбрать жидкость которая находится прямо в посудомойке.
Символ строгой вложенности. CSS ищет только среди детей, но не среди внуков.
комната №4
[type="Coffee"]
Все теги с атрибутом type="Coffee"
Селекторы могут искать не только классы. Квадратные скобки означают поиск среди атрибутов.
комната №5
.Liquid[type="Coffee"]
Выбрать только жидкий кофе
Еще пример составного селектора без пробелов. Будет выбран только тот элемент, который удовлетворяет всем требованиям.
комната №7
.Cup + .Cup
Буквально — выбрать чашку, перед которой стоит чашка
Символ плюс означает, что надо смотреть не внутри тега, а перейти к ближайшему соседу
